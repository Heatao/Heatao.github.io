maximum distance of 2 nodes in a tree

发表于 2019-10-08

Divide and Conquer

问题描述

​ Given a binary tree, suppose that the distance between two adjacent nodes is 1, please give a solution to ﬁnd the maximum distance of any two node in the binary tree.

​ Note: For BinaryTree, each node has three properties, a int value and two TreeNode pointers to children. You can assume that the INPUT is the root TreeNode of the tree. For example, the maximum distance of any two nodes in the below binary tree is 5.

解题思路

描述

​ 由于二叉树的特点，父阶段的深度为左右子树深度大者加一，显然，两节点最远距离必存在于某树的左右子树中（左右子树各一节点），则我们可以在遍历树的过程中，比较左右子树深度之和作为临时最大值max_tag，进行更新，求得最大值。

正确性验证

对于一个tree的节点而言，它所对应maxdis为当前节点（不一定是整个tree的root）的左子树深度和右子树深度之和，与之前节点留下的最大距离间的最大值。

这样定义的话，每个节点都会对应一个maxdis，所有的maxdis中最大的，即是所求值，这样需要访问每个节点一次。

时间复杂度

对于二叉树，每个节点值访问了一次，时间复杂度为O(n)
